About This Blog

This blog documents technical learnings from building data infrastructure at scale—systems processing billions of events daily across retail and financial services domains.

Topics covered:

  • Distributed Systems — Kafka, Spark, Flink, and the tradeoffs between them
  • Cloud Data Platforms — GCP, Azure, and multi-cloud architectures
  • Data-as-a-Service — API design for data products
  • Regulated Industries — Compliance, audit trails, and data governance
  • AI Infrastructure — RAG systems, vector databases, and LLM orchestration

The goal is practical knowledge—architecture decisions, code patterns, and lessons learned from production systems.
